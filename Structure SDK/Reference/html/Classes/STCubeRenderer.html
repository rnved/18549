STCubeRenderer Class Reference

Inherits from NSObject
Declared in StructureSLAM.h

Overview

Helper class to render a cuboid.

STCubeRenderer can render a wireframe outline of a cube, and also highlight the part of scene which fits in the given cube. This can be used to better visualize where the current cuboid scanning volume is located.
