Overview

Store the depth, color and camera pose information of a single frame.

– initWithColorCameraPose:colorFrame:depthFrame:error:

Initialize with required frame data and pose. depthFrame can be nil if the depth information is not needed.

Parameters

The GLKMatrix4 camera pose for this keyFrame.
colorFrame

The STColorFrame color buffer.
depthFrame

The STDepthFrame depth frame image.
error
