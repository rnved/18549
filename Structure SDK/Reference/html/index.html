Structure SDK Reference

Welcome, Structure App Developer!

The Structure SDK is made of the Structure Framework, sample applications to get you started, and a complete framework documentation generated from its headers.

You can find out more about Structure app development at: http://structure.io/developers.

Framework Overview

The Structure framework defines an Objective-C interface for developing applications leveraging the Structure Sensor on iOS devices.

It is split in two parts:

  • A low-level Sensor Controller layer, exposing raw depth & color stream access along with sensor information & status.
  • A high-level SLAM Engine, including 3D mapping, tracking and scanning features.

Sample Apps (Native)

The following sample apps have no non-standard dependencies (other than the Structure framework), and can be built and run with Xcode alone.

 

Viewer

The Viewer application is a simple stream viewer displaying the raw color camera stream, colored depth stream, and the estimated surface normal of each depth frame.

Scanner

The Scanner application is a minimal yet powerful mobile 3D scanner, covering a majority of the Structure framework functionality (high-level and low-level).

Room Capture

The Room Capture application demonstrates the acquisition of larger, textured environments. After a capture, the resulting mesh can be measured in true metric scale.

Unbounded Tracker

The Unbounded Tracker application demonstrates 6DOF positional tracking over an effectively infinite environment. The sample includes some assets from our released game S.T.A.R. OPS (available in the App store).

Unity Plugins

Two Unity packages are available to make it easy to get started with Structure SDK for developers who are familiar with Unity. Both packages use the native Structure SDK internally, and can be extended as needed by developers.

See the Unity folder for more, including Unity Plugins Overview.

StructureUnityAR

StructureUnityAR enables small-scene Augmented Reality experiences complete with occlusion and drift-free tracking within Unity. Two sample scenes, Fetch and Ball Physics are provided. See StructureUnityAR: Getting Started.

StructureUnityUBT

StructureUnityUBT enables Structure SDK’s unbounded 6-DoF positonal tracking (UBT) within Unity. Several sample scenes, Barebones, Example, MotionLogging, Stacker, SanFranciscoApartment, are provided. See StructureUnityUBT: Getting Started.

App Development

Developing applications for the Structure sensor requires Xcode 6 or above, freely available on the Apple developer site.

Important: The Structure framework requires the C++ runtime. In order to link it into your application, you need to:

  • Either #import from a .mm source.
  • Or append -lc++ to Xcode's other linker flags and define HAS_LIBCXX before #importing the framework headers.

Important: When creating a new application using the Structure SDK, the main Info.plist needs to contain an additional key "Supported external accessory protocols", with the following array of values:

  • io.structure.control
  • io.structure.depth
  • io.structure.infrared
Without this modification to the property list, the application will not be able to connect to the sensor. All sample applications have this key/value array. Check them out!
